# 📝 Story & Message Sharing Website

This project is a simple website where users can **write and share stories or messages**.

## 📖 About The Project

The goal of this project is to create a basic platform where people can express themselves by writing short stories or messages.
It was built to practice **front-end web development and page structure**.

## 🛠️ Technologies Used

* HTML5
* CSS3

## ✨ Features

* Users can write their own stories or messages
* Simple and clean design
* Easy to read layout

## 🎯 Purpose

This project was created to improve my **web design and coding skills** while building an interactive page concept.
